📋 DEPLOYMENT STEPS:

1. COMMIT & PUSH YOUR CHANGES
   ─────────────────────────────
   git add .
   git commit -m "Fix: Update for Streamlit Cloud deployment"
   git push origin main


2. REDEPLOY ON STREAMLIT CLOUD
   ─────────────────────────────
   Go to: https://share.streamlit.io/
   
   Option A: Automatic Redeployment
   • If already deployed, changes will auto-deploy (wait 2-3 min)
   
   Option B: Manual Reboot
   • Click "⋮" menu on your app
   • Select "Reboot app"
   
   Option C: Clear Cache & Redeploy
   • Click "⋮" menu
   • Select "Clear cache"  
   • Then "Reboot app"


3. VERIFY DEPLOYMENT
   ─────────────────────────────
   ✓ Check logs for "Processed dependencies!"
   ✓ No import errors for pdfplumber, docx2txt, scikit-learn
   ✓ App should load without errors
   ✓ Vector database features will be disabled (expected)

🐛 TROUBLESHOOTING:

ERROR: ModuleNotFoundError: No module named 'X'
SOLUTION: 
  • Clear cache and reboot
  • Check requirements.txt is committed
  • Wait 2-3 minutes for rebuild


ERROR: PDF extraction fails
SOLUTION:
  • Ensure packages.txt is committed
  • Check PDF is not password-protected


ERROR: App still showing old version
SOLUTION:
  • Hard refresh browser (Cmd+Shift+R / Ctrl+Shift+R)
  • Clear Streamlit cache from menu
  • Reboot app from Streamlit Cloud dashboard
